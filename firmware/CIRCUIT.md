# RC Car ESP32-C3 Super Mini 电路连接图

## 引脚分配

| ESP32-C3引脚 | 功能 | 连接目标 |
|--------------|------|----------|
| GPIO 6 | PWM输出 | 舵机信号线(橙/白) |
| GPIO 7 | PWM输出 | 电调信号线(橙/白) |
| GPIO 3 | ADC输入 | 电池电压检测(分压) |
| GND | 地线 | 共地 |
| 5V | 电源 | 可选,给ESP32供电 |

## ESP32-C3 Super Mini 引脚图

```
        ┌─────────────────┐
        │  ESP32-C3 Mini  │
        │   ┌───────┐     │
   5V ──┤   │ USB-C │     ├── GPIO5
  GND ──┤   └───────┘     ├── GPIO6  ──> 舵机信号
  3V3 ──┤                 ├── GPIO7  ──> 电调信号
GPIO4 ──┤    [BOOT]       ├── GPIO8
GPIO3 ──┤ <── 电池检测    ├── GPIO9
GPIO2 ──┤    [RST]        ├── GPIO10
GPIO1 ──┤                 ├── GPIO20
GPIO0 ──┤                 ├── GPIO21
        └─────────────────┘
```

## 电压检测电路

```
    2S电池 (+)
        │
       ┌┴┐
       │ │ 100KΩ
       └┬┘
        ├──────────> GPIO3 (ADC)
       ┌┴┐
       │ │ 47KΩ
       └┬┘
        │
       GND

分压比: 47/(100+47) = 0.32
最大检测电压: 3.3V / 0.32 = 10.3V
```

## 接线

```
舵机:
  - GND ────> ESP32 GND
  - +5V ────> 外部5V (电流大,不要接ESP32)
  - 信号 ───> GPIO6

电调:
  - GND ────> ESP32 GND
  - +5V ────> 可给ESP32供电 (BEC)
  - 信号 ───> GPIO7
```

## 注意事项

1. **ESP32-C3 只支持 BLE**，不支持经典蓝牙
2. **共地**: 所有GND必须连接在一起
3. **电源**: 舵机需要外部5V，电调BEC可给ESP32供电
4. **ADC**: GPIO3 支持 ADC，用于电池检测
